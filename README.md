## Complexity First Mindset

Har solution ke saath complexity sochna mandatory hai:

- Time Complexity: O(1), O(log n), O(n), O(n log n), O(n^2)
- Space Complexity: Auxiliary memory usage

Target: brute se optimal tak move karna aur reason clearly explain kar pana.

## How to Run the Code

### 1) Compile all Java files

```bash
javac BasicMath/*.java Hashing/*.java Sorting/*.java
```

### 2) Run any class

```bash
java BasicMath.GCD
java BasicMath.CountDivisors
java Sorting.BubbleSort
java Hashing.numbercount
```
