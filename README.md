# DSA Java

This repository contains Java implementations of basic DSA concepts with simple examples.

## Topics Covered

- Basic Math
  - Count digits
  - Reverse number
  - Palindrome number
  - Armstrong number
  - GCD (Euclidean algorithm)
  - Count divisors
  - Prime numbers
- Hashing
  - Number/character frequency counting
- Sorting
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort

## Project Structure

- `BasicMath/`
- `Hashing/`
- `Sorting/`

## How to Run

### 1) Compile all files

```bash
javac BasicMath/*.java Hashing/*.java Sorting/*.java
```

### 2) Run any class

Examples:

```bash
java BasicMath.GCD
java BasicMath.CountDivisors
java Sorting.BubbleSort
java Hashing.numbercount
```

## Goal

This repo is made for practice and revision of DSA fundamentals in Java.
